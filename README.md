
# Tinder Deblur

Tinder script to remove blur from photos.




# Installation

To run it you have to download the tampermonkey extension. https://www.tampermonkey.net/

Once installed you will have to open it and go to the section dashboard and click on Add +.

You paste the script and save it once saved, if it is not activated, activate it, enter the app and it would be done.
